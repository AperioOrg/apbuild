Installation Instructions

  1. Configure disks. My current configuration is: 
    /dev/sda1 — primary — almost full disk size — type: Linux.
    /dev/sda2 — extended — the rest of the disk size — type: Extended.
    /dev/sda5 — type: Swap.
          
    mkfs.ext4 /dev/sda1.
mkswap /dev/sda5.
  2. Mount root disk to /mnt.
  3. Run apinstall /mnt.
  4. Inside Aperio chroot, run apchroot.
  5. Configure locales and other personal stuff.
  6. Add an user.
  7. Add the user to the video, audio and input groups:
    usermod -a -G audio,video,input $USER
  8. Set a password for the user:
    passwd $USER
  9. Set a password for the root user:
    passwd root
  10. Setup GRUB:
    grub-install /dev/sda --root-directory=/ update-grub
  11. Exit Aperio chroot.
  12. Unmount everything:
    umount /mnt/* umount /dev/sda1
  13. Reboot
  14. Done!

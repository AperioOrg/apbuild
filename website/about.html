Aperio started in 2020. It was meant to be just a Debian GNU/Linux installation script, but after some modifications I decided to make a Debian-based distribution. My goals were stability and the possibility to use it in Desktops and servers.

Features

  • Stable.
  • No default window manager.
  • No default desktop manager.
  • No default display manager.
  • Easy installation.
